Avoid Using Volatile (AvoidUsingVolatile)

'volatile' keyword 가 사용되었을 경우. 'volatile' keyword 는 주로 java application 을 최적하기 위해 사용되므로, Java Memory Model 에 대해 잘 알고 있어야 한다. 게다가 그 action 의 범위는 다소 잘못 알려져 있다. 그러므로 유지 가능성과 이식성을 위해서 'volatile' keyword 는 사용하지 않는 것이 좋다. 실제적으로 platform, 운영체제, jvm 에 따라 volatile 이 기대한 대로 작동하지 않는 경우가 많다.
