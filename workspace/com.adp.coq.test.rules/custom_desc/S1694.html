An abstract class should have both abstract and concrete methods (S1694)

abstract 클래스는 서브 클래스에서 반드시 구현되어야 할 메쏘드를 정의함과 동시에 상속가능한 behavior 를 제공한다. 단지 Instantiation 를 피하기 위해 abstract 메쏘드 없이 abstrace 클래스를 만들었다면, 이는 private constructor 를 가진 concrete 클래스로 변경되어야 한다. 또, abstract 메쏘드는 있지만 상속할만한 behavior 를 가지고 있지 않은 클래스는 interface 로 변경한다.
